# Практическая работа 15: Бинарное дерево поиска

## Инструкция для запуска и проверки методов
### Запустите программу:

```bash
python3 main.py
```
### Появится меню:
```text
=== BST DEMO ===
1. INSERT  - добавить элемент
2. SEARCH  - найти элемент  
3. DELETE  - удалить элемент
4. HEIGHT  - высота дерева
5. IS_BALANCED - проверка баланса
6. RESET   - сбросить дерево
7. EXIT    - выход
```

### Введите цифру от 1 до 7:


#### INSERT (добавление):

```text
Command: 1
Key: 25
Value (Enter for none): my_value
Added 25
```
#### SEARCH (поиск):

```text
Command: 2  
Key to search: 30
Found: val_30
```
#### DELETE (удаление):

```text
Command: 3
Key to delete: 30
Deleted 30
```
#### HEIGHT (высота):

```text
Command: 4
Tree height: 3
```
#### IS_BALANCED (баланс):

```text
Command: 5
Balanced: True
```
#### RESET (сброс):

```text
Command: 6
Tree reset to demo state
```
